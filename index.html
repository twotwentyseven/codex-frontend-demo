Stylesheets have been split into individual files for each component, some components require multiple stylesheets for example Popout cart will require login stylesheet, you could however omit this login stylesheet and add custom styles for only the popout cart. There is a base stylesheet which holds all of the css variables and global styles, which should be place within the head of project.
